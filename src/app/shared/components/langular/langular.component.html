Start an application

There's a tool named CLI, it is the easy way to mange an Angular application, so we will install CLI, after that create our aplication. Go to terminal and type:
$> npm install -g @angular/cli
$> ng new my-dream-app
$> cd my-dream-app
$> ng serve
